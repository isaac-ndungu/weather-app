# Weather Dashboard App

## Project Brief
This project is a simple weather application built using HTML, CSS, and JavaScript. It integrates with the OpenWeather API to fetch and display real-time weather data for a default city (Nairobi) and any user-searched city.

## Contributor
Isaac Ndungu

## Core Features
- Displays current weather for Nairobi on page load
- Allow users to search for any city
- shows: 
    - City name and country
    - Weather description
    - temperature
    - Humidity
    - wind speed
    - visibility


## Technologies Used

- HTML5
- CSS3
- TailwindCSS
- Javascript
- OpenWeather API

## Usage Instructions

1. clone or download the repository
2. Open index.html in a browser.
3. Enter a city name and click search (Ensure internet connection is active).
